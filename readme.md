## Instalation

Clone this project
```
$ git clone git@github.com:ncasich/vue.js.git
```
Check the branch you need
```
$ git checkout 'selected_branchname'
```
Install modules needed for this branch
```
$ npm install
```
Build js & styles
```
$ npm run dev
```
